# youtube-clone
A responsive You Tube clone created using HTML and CSS


##Features

 -Responsive design using media queries
 
 -Navigation sidebar
 
 -Video grid layout using CSS grid
 
 -Header with search bar
 
 -Hover effects
 
 -Position properties for layout alignment
 
 -Box shadows and rounded corners
 
 
##Technologies Used

 -HTML5
 
 -CSS3
